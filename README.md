# 💪 HyperLift



HyperLift is a full-stack fitness web application built with **React**, **Spring Boot**, and **MySQL**. It allows users to explore exercises, follow workout plans, and provides an admin dashboard for managing exercises and workout content.



## 🚀 Features



* User Authentication & Authorization

* Role-Based Access Control

* Exercise Management

* Workout Plan Management

* RESTful API

* Responsive User Interface



## 🛠️ Tech Stack



**Frontend**



* React

* TypeScript

* Bootstrap

* Axios



**Backend**



* Java

* Spring Boot

* Spring Security

* Spring Data JPA

* Hibernate



**Database**



* MySQL



## ⚙️ Getting Started



### Backend



```bash

cd hyperlift-backend

./mvnw spring-boot:run

```



### Frontend



```bash

cd hyperlift-frontend

npm install

npm run dev

```
